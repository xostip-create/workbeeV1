# **App Name**: E&F WorkBee

## Core Features:

- Home Screen Display: Display the app name and navigation options upon loading the application.
- Find a Worker: Provide a visual option to navigate to the 'Find a Worker' page.
- Find a Shop: Provide a visual option to navigate to the 'Find a Shop' page.
- Post a Job: Provide a visual option to navigate to the 'Post a Job' page.

## Style Guidelines:

- Primary color: Saturated blue (#4285F4) to convey trust and reliability, essential for connecting workers and employers.
- Background color: Light blue (#E3F2FD), a very desaturated shade of the primary color.
- Accent color: Green (#34A853), analogous to blue, for calls to action, to signal opportunity.
- Body and headline font: 'PT Sans', a sans-serif font, is suitable for both headlines and body text.
- Use clean, modern icons to represent each main option: 'Find a Worker', 'Find a Shop', and 'Post a Job'.
- The home screen layout should be simple and intuitive, focusing on clear navigation.
- Subtle animations can be used when transitioning between screens.
